malDecision

A tool for data-driven decision-making to manage infectious diseases in environments with varying levels of urbanization.

malDecision used a machine learning method to prioritize the determinants. This tool helps visualize the spatial distribution of the region of interest and the hierarchy of the determinant post-analysis. Moreover, the user can download a custom report that could guide him in making informed decisions based on the disease trajectory.

The flexible and interactive malDecision tool was created to assist practitioners and decision-makers in adopting a data-driven strategy for the control of infectious diseases in their countries.

This tool is outlined in the peer-reviewed publication available.

Privacy Note

The program is offered in two forms: a standalone version that can be used in environments with modest resources and a web app that can be installed on a Shiny server. malDecision is a desktop application downloaded. It iterates over the loaded notes file while in use but does not store or save any data.
