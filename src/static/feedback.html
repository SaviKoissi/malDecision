Feedback

We would appreciate any feedback or ideas on malDecision tool. Please send an email to merveillekoissi.savi@gmail.com.

When reporting a problem, please include the following information:

  • What type of problem did you encounter?
  • Where did you encounter the problem; i.e. which tab was selected?
  • Did the problem occur systematically or randomly; i.e. were you able to reproduce the problem?
  • A screenshot of the Global Vaccination Survey Analyzer window displaying the problem.

If you discover a security issue, we kindly ask you to contact us privately by e-mail so we have enough time to investigate and fix the issue.

Thank you for your interest!
