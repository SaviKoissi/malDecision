malDecision

A Data-driven decision-making tool to manage infectious diseases with different degrees of urbanization.

The malDecision tool is a versatile and interactive application designed to help help-practitioners and decision makers take a data-driven approach to the management of infectious diseases in their countries.

Read the paper

Users upload a CSV file with the following header: |country|Region|District|long|lat|cluster|classifiers|Output . The front-end, graphical web interface of the tool is developed using Shiny, a web application framework for R. The malDecision tool generates an analysis report for the user to download.

The application is available in a standalone version suitable for use in resource-limited settings, and as a web app to be deployed on a Shiny server.
