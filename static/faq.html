Frequently Asked Questions

Please refer to the “Reference Manual” for more details. The manual is available for download here.

The malDcision tool is developed for use by public health field practitioners supporting the management of infectious diseases throughout countries with different degrees of urbanization. The tool is designed so that it can be used by a practitioner with no or limited knowledge of statistical analysis or software programming. The intention is to simplify analysis to allow end users to focus on interpretation and use of the data collected.

The tool is designed with resource limited settings in mind. Internet is needed to download the software version but not to run it. Once downloaded, the software can be used offline.

Data are stored locally on your computer. Data is not uploaded to a cloud or online server. You are not agreeing to share data by using the malDecision tool.
